🔴 EXP 5 – TWO LEDS USING RTOS
✅ Algorithm
Start
Initialize system and FreeRTOS
Create Task 1 (LED1 control)
Create Task 2 (LED2 control)
Start scheduler
Task 1:
Set LED1 HIGH
Delay
Set LED1 LOW
Delay
Task 2:
Set LED2 HIGH
Delay
Set LED2 LOW
Delay
Scheduler switches between tasks
Repeat continuously
Stop

🔴 EXP 6 – LED BRIGHTNESS (PWM)
✅ Algorithm
(A) WITH RTOS
Start
Initialize FreeRTOS
Create task for LED brightness control
Start scheduler
In task:
Initialize brightness = 0
Increase brightness gradually
Apply PWM using analogWrite()
Delay
Reset when max reached
Repeat continuously
Stop
(B) WITHOUT RTOS
Start
Configure LED pin as OUTPUT
Initialize brightness variable
Enter loop
Increase brightness
Apply PWM
Add delay
Decrease brightness
Repeat steps 5–8
Stop

🔴 EXP 7 – BINARY SEMAPHORE
✅ Algorithm
Start
Initialize FreeRTOS
Create binary semaphore
Initialize semaphore as available
Create multiple tasks
Start scheduler
For each task:
Request semaphore (xSemaphoreTake)
Enter critical section
Execute task
Release semaphore (xSemaphoreGive)
Delay
Scheduler switches tasks
Repeat continuously
Stop

🔴 EXP 8 – TASK PRIORITY SCHEDULING
✅ Algorithm
Start
Initialize FreeRTOS
Create multiple tasks
Assign different priorities to each task
Start scheduler
Scheduler selects highest priority task
Execute selected task
If higher priority task arrives → preempt current task
Continue scheduling based on priority
Repeat continuously
Stop

🔴 1. ULTRASONIC SENSOR (HC-SR04)
✅ Algorithm
Start
Initialize TRIG pin as OUTPUT
Initialize ECHO pin as INPUT
Initialize serial communication
Enter infinite loop
Set TRIG LOW for stabilization
Send HIGH pulse (10 µs) to TRIG
Wait for ECHO pin to go HIGH
Measure time duration of ECHO signal
Calculate distance:
👉 Distance = (Time × Speed of Sound) / 2
Display distance on serial monitor
Apply delay
Repeat steps 6–12
Stop

🔴 2. GAS SENSOR (MQ Series)
✅ Algorithm
Start
Connect gas sensor output pin to Arduino input pin
Initialize serial communication
Set sensor pin as INPUT
Enter infinite loop
Read sensor value (analog/digital)
Compare value with threshold
If value > threshold → Gas detected
Else → No gas detected
Display result on serial monitor
Apply delay
Repeat steps 6–11
Stop
